# Language Alias Test

This document tests various language aliases for syntax highlighting.

## C# Aliases

### Using `cs` alias
```cs
public class Foo
{
    public int Bar { get; set; }
}
```

### Using `c#` alias
```c#
public class Foo
{
    public int Bar { get; set; }
}
```

### Using `dotnet` alias
```dotnet
public class Foo
{
    public int Bar { get; set; }
}
```

### Using canonical `csharp`
```csharp
public class Foo
{
    public int Bar { get; set; }
}
```

## CoffeeScript Aliases

### Using `coffee` alias
```coffee
class Foo
{
    var z = 5;
    const abc = "3";
}
```

### Using canonical `coffeescript`
```coffeescript
class Foo
{
    var z = 5;
    const abc = "3";
}
```

## JavaScript Aliases

### Using `js` alias
```js

    return "world";
```

### Using canonical `javascript`
```javascript
const x = 5;
function hello() {
    return "world";
}
```
