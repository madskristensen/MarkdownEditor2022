# npm Quick Reference

## One-time Setup
```bash
npm install
```

## Update Mermaid
```bash
npm run update-mermaid
```

## Manual Copy (if needed)
```bash
npm run copy-mermaid
```

## Check for Updates
```bash
npm outdated
```

## Check for Vulnerabilities
```bash
npm audit
```

## View Installed Mermaid Version
```bash
npm list mermaid
```

## Remove npm Artifacts (if needed)
```bash
rm -r node_modules package-lock.json
npm install
```
